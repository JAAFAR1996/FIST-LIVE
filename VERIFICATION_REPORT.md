### 1. اختبار نظام المفضلة
```
1. افتح صفحة المنتجات
2. ابحث عن أيقونة القلب ❤️ على أي منتج
3. اضغط عليها
4. تحقق من ظهور notification
5. اذهب إلى Navbar
6. ابحث عن أيقونة المفضلة مع عداد
7. اضغط عليها

```

**النتيجة المتوقعة:** يجب أن ترى جميع المنتجات المفضلة

---

### 2. اختبار خصم Journey
```
1. اذهب إلى /journey
2. أكمل جميع الخطوات
3. في الخطوة الأخيرة، ابحث عن:
   - Badge أصفر "خصم 10%"
   - كود: JOURNEY2024
   - زر "طبق الكود"
```

**النتيجة المتوقعة:** يجب أن ترى قسم الخصم واضحاً

### 3. اختبار الثيمات
```
1. ابحث عن زر تبديل الثيم في Navbar

   - 🌙 داكن
   - 💻 النظام
3. جرب كل خيار
```

**النتيجة المتوقعة:** لا توجد خيارات أخرى

### 4. اختبار Favicon

1. افتح الموقع في Chrome/Firefox

```

### 5. اختبار أزرار الإغلاق
```
1. افتح أي نافذة منبثقة (مثل Fish Details)
2. انظر إلى زر X في الزاوية العليا
3. يجب أن يكون:
   - كبير (40x40px)
   - دائري
   - له حدود واضحة
   - يتغير عند hover
4. جرب:
   - النقر على زر X
   - النقر خارج النافذة
```

**النتيجة المتوقعة:** النافذة تغلق بسهولة

```
