## 🚀 كيفية البدء

### 1. تشغيل الخادم

```bash
# في وضع التطوير
pnpm dev

# أو في وضع الإنتاج
pnpm build
pnpm start
```

### 2. تسجيل الدخول

افتح المتصفح وانتقل إلى:
```
http://localhost:5000/admin
```

استخدم بيانات الاعتماد التالية:
- **البريد الإلكتروني**: jaafarhabash@outlook.com
- **كلمة المرور**: (كلمة المرور الموجودة لهذا الحساب)

### 3. إدارة المنتجات

من لوحة التحكم يمكنك:
- ➕ **إضافة منتج جديد**: اضغط على زر "إضافة منتج جديد"
- ✏️ **تعديل منتج**: اضغط على أيقونة القلم
- 🗑️ **حذف منتج**: اضغط على أيقونة سلة المهملات
- 🔍 **البحث**: استخدم مربع البحث في الأعلى

## 📋 الأوامر المتاحة

```bash
# تشغيل إعداد المسؤول (إذا احتجت لإعادة الإعداد)
DATABASE_URL='your_url' pnpm admin:setup

# فحص قاعدة البيانات
DATABASE_URL='your_url' pnpm admin:check

# فحص الأخطاء البرمجية
pnpm check

# تشغيل الاختبارات
pnpm test
```
